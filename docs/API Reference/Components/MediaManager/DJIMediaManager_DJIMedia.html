      Fetch Thumbnail
      method
      fetchThumbnailWithCompletion
      method fetchThumbnailWithCompletion
      Header:DJIMediaFile.h
      SDK Key:NA
      Doc Key:DJIMediaManager_DJIMedia_fetchThumbnail
      Description:

      Fetches this media's thumbnail with a resolution (99 x 99) from the SD card. This method can be used to fetch either a photo or a video, where the first frame of the video is the thumbnail that is fetched. It is not available if the media type is Panorama.

      Precondition:
      The camera mode is MediaDownload mode.

      Post Condition:
      This method will start to download the media thumbnail in the SD card. The content can be videos or images.

      Input Parameters:
      SOURCE_DIFFERENT blockCompletion block.
      Fetch Media Data
      method
      fetchMediaDataWithCompletion
      method fetchMediaDataWithCompletion
      Header:DJIMediaFile.h
      SDK Key:NA
      Doc Key:DJIMediaManager_DJIMedia_fetchMediaData
      Description:

      Fetches this media's data from the SD card. The difference between fetching the media data and fetching the thumbnail is that fetching the thumbnail will return a low-resolution image of the actual picture, while fetching the media data will return all data for a video or image.

      Input Parameters:
      NSData *_Nullable dataThe media's data.
      BOOL *_Nullable stopThe BOOL value.
      NSError *_Nullable errorError retrieving the value.
      SOURCE_DIFFERENT blockCompletion block to receive the result.
      Fetch Preview Image
      method
      fetchPreviewImageWithCompletion
      method fetchPreviewImageWithCompletion
      Header:DJIMediaFile.h
      SDK Key:NA
      Doc Key:DJIMediaManager_DJIMedia_fetchPreviewImage
      Description:

      Fetch media's preview image. The preview image is a lower resolution (960 x 540) version of a still picture or the first frame of a video. The DJIMediaType of this media object should be DJIMediaTypeJPEG. It is not available if the media type is DJIMediaTypePanorama.

      Input Parameters:

      NSError *_Nullable errorError retrieving the value.
      SOURCE_DIFFERENT blockCompletion block to receive the result.
