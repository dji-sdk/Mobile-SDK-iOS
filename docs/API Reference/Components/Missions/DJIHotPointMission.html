      class DJIHotpointMission

      Header:DJIHotpointMission.h
      Inherits From:DJIMission
      Description:

      These class represents a Hotpoint mission. In a Hotpoint mission, the aircraft will repeatedly fly circles of a constant radius around a specified point called a Hot Point. The user can control the aircraft to fly around the Hotpoint with a specific radius and altitude. During execution, the user can also use the physical remote controller to modify its radius and speed. It is not supported by Mavic Pro when using WiFi connection. It is not supported by Spark.

      Header:DJIHotpointMission.h
      Description:

      Aircraft's heading while flying around the hot point. It can be pointed toward or away from the hot point, forward or backward along its flight route, or can be manually controlled by the remote controller.

      Header:DJIHotpointMission.h
      Description:

      Sets the altitude of the hot point orbit with a range [5,500] meters. The altitude is relative to the ground altitude from which the aircraft took off.

      Radius
      property
      radius
      property radius
      Header:DJIHotpointMission.h
      Description:

      Sets the circular flight path radius with which the aircraft will fly around the hotpoint. The value of this property should be in the range of [DJIHotpointMinRadius, DJIHotpointMaxRadius] meters.

      Angular Velocity
      property
      angularVelocity
      property angularVelocity
      Header:DJIHotpointMission.h
      Description:

      Sets the angular velocity in degrees/s of the aircraft as it orbits the hot point. The default value is 20 degrees/s. The maximum angular velocity allowed is relative to the orbit radius.
