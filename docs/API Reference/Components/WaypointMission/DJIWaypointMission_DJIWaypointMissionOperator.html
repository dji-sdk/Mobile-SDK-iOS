      Description:

      List of all possible events that occur during the life cycle of a waypoint mission

      Enum Members:
      DJIWaypointMissionEventUnknownUnknown event.
      DJIWaypointMissionEventInitializedThe waypoint abstraction is initialized and recovered from the disconnection.
      DJIWaypointMissionEventEnterNavigationModeThe aircraft enters navigation mode.
      DJIWaypointMissionEventExitNavigationModeThe aircraft exits navigation mode. If there is executing mission, the mission will be intterupted (state changes from executing to NotReady).
      DJIWaypointMissionEventRCModeChangedUser changes the Remote Controller switch. If there is executing mission, the mission will be intterupted (state changes from executing to NotReady if multi-mode is enabled or state changes from executing to ReadyToUpload if multi-mode is disabled).
      DJIWaypointMissionEventSimulationOffSimulation mode is changed from on to off.
      DJIWaypointMissionEventDisconnectedThe connection is broken.
      DJIWaypointMissionEventUploadFailedError happens during the upload.
      DJIWaypointMissionEventUploadProgressUpdateThe upload progress is updated.
      DJIWaypointMissionEventUploadDoneThe mission is uploaded to the aircraft completely.
      DJIWaypointMissionEventDownloadFailedError happens during the download.
      DJIWaypointMissionEventDownloadProgressUpdateThe download progress is updated. User can use members in DJIWaypointDownloadProgress to know the detail progress.
      DJIWaypointMissionEventDownloadDoneThe mission is uploaded to the aircraft completely. User can get the complete mission (data member) in DJIWaypointDownloadProgress.
      DJIWaypointMissionEventExecutionStartedThe mission execution has started.
      DJIWaypointMissionEventExecutionStartFailedStarting of the execution failed.
      DJIWaypointMissionEventExecutionPausedThe mission was paused
      DJIWaypointMissionEventExecutionPauseFailed
      DJIWaypointMissionEventExecutionStoppedThe mission was stopped
      DJIWaypointMissionEventExecutionStopFailedStopping the mission failed
      DJIWaypointMissionEventExecutionResumedA previously paused execution was resumed
      DJIWaypointMissionEventExecutionResumeFailedResuming failed.
      DJIWaypointMissionEventExecutionProgressUpdateThe execution progressed and generated an update.
      DJIWaypointMissionEventExecutionInterruptedThe mission execution was interrupted.
      DJIWaypointMissionEventExecutionFinishedThe mission execution finished. Normal completion.

      Doc Key:DJIWaypointMission_DJIWaypointMissionOperator_DJIWaypointMissionState
      Description:

      List of all states a mission may go through during its life cycle.

      Enum Members:
      DJIWaypointMissionStateUnknownUnknown State.
      DJIWaypointMissionStateNotReadyThe current state is not ready for uploading a mission.
      DJIWaypointMissionStateReadyToUploadIt is OK to start uploading a waypoint mission.
      DJIWaypointMissionStateReadyToRetryUploadIt is OK to retry the uploading from the previous timeouted waypoint.
      DJIWaypointMissionStateUploadingThe uploading is started successfully. Detail information for each waypoint is being uploaded one by one.
      DJIWaypointMissionStateReadyToExecuteWaypoint mission is uploaded completely and the aircraft is ready to start the execution.
      DJIWaypointMissionStateExecutingThe execution is started successfully.
      DJIWaypointMissionStateExecutionPausedWaypoint mission is paused successfully. User can call resume to continue the execution.
      DJIWaypointMissionStateExecutionStoppingThe stop command has been called and we didn't receive the ACK from the aircraft yet. It is NOT allowed to call another command in this state. The next state is either disconnected or the previous state (executing or paused).
